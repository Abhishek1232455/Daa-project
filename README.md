# 🚀 Algorithm Design Playground (C++ Backend)

A high-performance algorithm visualizer built with **React**, **Node.js**, and a native **C++ backend engine** for fast and efficient algorithm execution.

---

## ✨ Features

* ⚡ **C++ Engine Performance**
  Sorting and searching algorithms are executed in native C++ for maximum speed.

* 🎨 **Vivid Visualization**
  Smooth real-time animations powered by Framer Motion with a sleek **Black & Crimson UI theme**.

* 🔢 **Custom Inputs**
  Enter your own arrays and search target values dynamically.

* 📝 **Detailed Execution Logs**
  Step-by-step algorithm breakdown generated directly from the C++ engine.

* 🖥️ **Modern Responsive UI**
  Sidebar-driven interface with Space Grotesk typography and live backend status monitoring.

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Abhishek1232455/Daa-project.git
cd Daa-project
```

### 2️⃣ Backend Setup

```bash
cd backend
npm install
npm run build:cpp
npm start
```

Backend runs on:

```bash
http://localhost:3001
```

### 3️⃣ Frontend Setup

From root directory:

```bash
npm install
npm run dev
```

Frontend runs on:

```bash
http://localhost:5173
```
