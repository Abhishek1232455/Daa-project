# Algorithm Design Playground (C++ Backend)

A high-performance algorithm visualizer built with **React** and a **Node.js + C++** backend. 

## ✨ Features
- **C++ Engine**: Algorithms (Sorting & Search) are computed in native C++ for maximum performance.
- **Vivid Visualization**: Smooth Framer Motion animations with a distinct **Black & Crimson** theme.
- **Custom Inputs**: Define your own arrays and search for specific target values.
- **Detailed Logs**: Step-by-step logic breakdown generated by the C++ engine.
- **Modern UI**: Sidebar-driven layout with Space Grotesk typography and real-time backend status monitoring.

## 🛠️ Tech Stack
- **Frontend**: React, Vite, Framer Motion, Lucide Icons.
- **Backend Server**: Node.js, Express.
- **Algorithm Core**: C++ (Compiled binary).

## 🚀 Getting Started

### 1. Backend Setup
```bash
cd backend
# Compile the C++ engine
npm run build:cpp
# Start the Express server
npm start
```

### 2. Frontend Setup
```bash
# From the root directory
npm install
npm run dev
```

The application will be available at `http://localhost:5173`. Ensure the backend is running on `http://localhost:3001` for the C++ engine to connect.

## 📦 Project Structure
- `/src`: React frontend logic and visualizer components.
- `/backend`: Node.js server that interfaces with the C++ binary.
- `/backend/src/main.cpp`: The core C++ algorithm implementation.
